# ba_diploma
Business Analyst Diploma Project
Разработка решения для модернизации интернет-магазина строительных товаров «Переделкино»

## О проекте

Этот репозиторий содержит мой дипломный проект по курсу «Бизнес-Аналитик». 

**Тема проекта:** разработка требований к решению для модернизации интернет-магазина строительных товаров «Переделкино».

В ходе работы я провела полный цикл бизнес-анализа: от обследования текущих процессов до формирования готовых требований к решению и презентации результатов заказчику.

---

## Цель работы

Проанализировать текущее состояние интернет-магазина «Переделкино», выявить точки роста и сформулировать требования к решению, которое позволит:
- закрыть выявленные проблемы
- улучшить качество бизнес-процессов
- повысить эффективность работы магазина

##  Что в итоге

### Часть 1. Отчёт об обследовании

В отчёте описано текущее состояние процессов интернет-магазина «Переделкино»:
- анализ бизнес-процессов (AS IS)
- выявленные проблемы и болевые точки
- зоны для улучшения и потенциальные точки роста
- рекомендации по развитию

### Часть 2. Документ с требованиями и презентация

На основе проведённого обследования разработан:
- полный документ с требованиями к решению
- видеопрезентация для заказчика с обоснованием предлагаемых изменений

---

##  Используемые методологии и инструменты

- BPMN / UML для моделирования процессов
- анализ стейкхолдеров
- техники сбора и приоритизации требований
- AS IS / TO BE анализ
- визуализация данных и процессов
